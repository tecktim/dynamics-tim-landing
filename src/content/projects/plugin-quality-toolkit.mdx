**Bachelor-Thesis Projekt** zur Automatisierung der Einrichtung von Code Quality & Security Checks in der Projekt- und Produktentwicklung. 

Das Framework umfasst:
- Statische Codeanalyse und Security-Scanning
- Automatisierte Quality Gates in CI/CD-Pipelines
- Automatisches Rollback bei Quality- oder Security-Verstößen
- Integration in Azure DevOps für verbesserte Entwicklungsprozesse
